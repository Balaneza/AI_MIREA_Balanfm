# Данные проекта

Используется открытый датасет UCI **APS Failure at Scania Trucks**.

Файлы:

- `raw/aps_failure_training_set.csv` — 60 000 строк train, 1 000 `pos`.
- `raw/aps_failure_test_set.csv` — 16 000 строк test, 375 `pos`.
- `raw/aps_failure_description.txt` — описание датасета, лицензии и cost metric.
- `demo_payload.json` — пример JSON для `POST /predict-failure`.

Особенности формата:

- первые 20 строк CSV содержат текстовую шапку UCI, заголовок таблицы начинается после неё;
- целевая переменная называется `class`;
- пропуски записаны строкой `na`;
- все 170 признаков анонимизированы и являются числовыми сенсорными/агрегированными признаками.

Данные открытые и обезличенные. Персональные данные и секреты в этой папке не используются.
